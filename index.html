Linear Regression with Gradient Descent (and Flowers!)

This simulation is designed to teach you about gradient descent. Click on the green below to create points in a dataset. Once you have more than two (2) points, you create a regression line. As more points are added, the program uses gradient descent to find the regression line of those points by adjusting the m and b values in the line equation y=mx+b . Play around with this!
